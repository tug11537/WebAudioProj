# Interactive Sound Space

An interactive web-based sound instrument built with Tone.js.

Move your mouse across the screen to explore timbral space:

- **X-axis:** controls filter cutoff (brightness)
- **Y-axis:** controls reverb depth and subtle pitch variation

## Features

- Dual-oscillator voice (sawtooth + sine fifth layer)
- Lowpass filter with Q shaping
- Long-form reverb space
- LFO-based filter modulation for continuous motion
- Click-free start/stop transport
- Real-time gesture mapping
- Adjustable UI controls (master, cutoff range, reverb, LFO rate)

## Technologies Used

- Tone.js
- Web Audio API
- HTML / CSS / JavaScript

## How to Run

1. Clone the repository
2. Open `index.html` in a browser
3. Click **Start**
4. Move your mouse to shape the sound

## Concept

This project explores gesture-based synthesis in a browser environment.  
The goal was to create a playable sound space where motion maps to spectral and spatial transformations in real time.
